2.Choice of Plausible Alternatives (COPA)

Paper: https://asgordon.github.io/publications/AAAI-SPRING11A.PDF

Data Distribution: - 1000 groups - each group has 2 relations, one is correct and the other is false. - only 2 types of relation (cause, effect)

Data Format:

3. e-CARE

Paper: https://arxiv.org/pdf/2205.05849

Data Distribution: - 21,324 groups (7,617 groups of cause relation and 7,311 groups of effect relation) - each group has 2 relations (one is correct and the other is false) and a explanation.

Data Format:
